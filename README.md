# AI-For-Beginners

Welcome to **AI-For-Beginners**, a beginner-friendly website and book project designed to introduce anyone to the fascinating world of Artificial Intelligence (AI). This project is built using **Docusaurus**, and the content of the book has been written with the help of **Claude** and **OpenAI's Speckit**.
